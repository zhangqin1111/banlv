# Scope Matrix

Use this file to decide whether a request belongs in the current MVP.

## Product identity that must stay

- Treehole
- Mood Weather
- Three emotional mode scenes
- Blind Box Lite
- My momo
- Basic growth feedback
- Safety, delete, report, privacy, and age notice

## P0 modules

- Home entry flow
- Mood check-in and Mood Weather result
- Treehole chat with SSE
- Three mode scenes:
  - `joy_mode` -> `light_drift`
  - `low_mode` -> `cloud_breathing`
  - `anger_mode` -> `bubble_release`
- Blind Box Lite
- My momo page with stage feedback
- Records and recent history
- Safety block and report/delete flows

## P1 polish

- Better scene transitions
- Better particles and blur
- More blind-box card variants
- Richer history visualizations
- Better growth celebration states

## Not now

- 99 levels
- focus forest
- community
- tarot, MBTI, horoscope
- voice input or TTS
- cross-platform sync beyond the first app target
- store, skins, item economy

## Decision rule

If a request does not strengthen one of the product-identity modules above, it is probably `not-now`.
