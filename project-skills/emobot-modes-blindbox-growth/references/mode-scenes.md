# Mode Scenes

Use this file for the three emotional micro-interactions.

## Shared rules

- 45 to 90 seconds
- one clear input pattern
- one clear calming finish
- no scoreboards, levels, or failure state

## `joy_mode` -> `light_drift`

- mood: bright and upward
- input: tap or drag to gather light particles
- finish: scene brightens and momo glows slightly

## `low_mode` -> `cloud_breathing`

- mood: slow and airy
- input: follow a breathing rhythm for three rounds
- finish: clouds open, pace softens, momo settles nearby

## `anger_mode` -> `bubble_release`

- mood: safe release, then calm
- input: tap or swipe away pressure bubbles
- finish: motion slows and the scene closes with one grounding breath
