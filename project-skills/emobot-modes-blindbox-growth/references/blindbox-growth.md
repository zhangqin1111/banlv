# Blind Box and Growth

Use this file for draw behavior and stage logic.

## Blind Box Lite rules

- one draw returns one card
- cards come from a template library
- card types:
  - comfort
  - action
  - reframe
- AI may polish wording, but should not invent the full product logic

## Growth points

Suggested deterministic awards:

- check-in complete: `+1`
- Treehole meaningful completion: `+2`
- mode scene complete: `+2`
- Blind Box Lite draw: `+1`

## Growth stages

- `seed`
- `bloom`
- `glow`

## Growth presentation

- show gentle progress
- show stage name
- avoid RPG or gacha language
