# Motion and Visual Style

Use this file for animation and visual polish decisions.

## Motion rules

- Soft, short, low-pressure.
- Prefer opacity, scale, offset, and glow over complex simulation.
- Keep animation smooth on a mid-tier phone.
- If performance drops, reduce particles before changing layout.

## momo motion

- idle float
- slow breathing
- occasional blink
- slight lean toward active content
- warm glow on successful completion

## Mode scene mood

- `light_drift`: bright, upward, airy
- `cloud_breathing`: slow, wide, calm
- `bubble_release`: safe release, quick input, calm finish

## Forbidden cues

- hard shake
- violent burst
- harsh red-black contrast
- arcade-like score flashes
- victory or failure language
