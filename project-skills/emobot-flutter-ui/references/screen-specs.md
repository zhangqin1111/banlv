# Screen Specs

Use this file when building or editing pages.

## Home

- Lead with momo hero and one calm prompt.
- Show the four core entry cards.
- Surface the latest emotion summary if available.

## Mood Weather

- Capture mood type, intensity, and one short line.
- Return empathy text plus invitation cards.
- Never force the user into a mode scene.

## Treehole

- Stream replies.
- Keep composer lightweight.
- Support loading, interrupted stream, and safe fallback states.

## Mode scenes

- Each mode should finish in 45 to 90 seconds.
- Show a clear exit and gentle completion state.
- Persist a summary result back to records and growth.

## Blind Box Lite

- One draw gives one card.
- Cards should read fast and feel comforting.
- Support save, close, and draw-another-if-allowed states.

## My momo

- Show current stage, growth points, and recent activity.
- Keep the page reflective, not game-like.
