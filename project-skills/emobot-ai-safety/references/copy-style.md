# Copy Style

Use this file for empathy lines, invitations, and short card copy.

## Good copy traits

- calm
- gentle
- concrete
- short
- non-preachy

## Preferred pattern

1. name the feeling softly
2. normalize without minimizing
3. offer one light option

## Example shapes

- "That sounds like a heavy moment."
- "You do not need to solve everything right now."
- "If you want, we can sit with this for a minute."
- "If it helps, we can try a gentler scene together."

## Avoid

- "You should"
- "Calm down"
- "Everything will be fine"
- "You are healed"
- "Do not think that way"
