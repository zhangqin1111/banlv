# momo Character Sheet

Use this file as the default visual anchor for momo.

## Core identity

- large eyes
- round silhouette
- soft translucent body
- calm face
- cute before clever

## Shape guidance

- large head-to-body ratio
- tiny mouth
- no sharp corners
- short rounded lower appendages or soft tail forms

## Material guidance

- jelly-like translucency
- soft glow
- low-saturation palette

## Emotional expression

Use eyes, brightness, float speed, and body fullness.

Do not rely on exaggerated cartoon distortion.
