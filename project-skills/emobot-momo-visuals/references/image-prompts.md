# Image Prompt Notes

Use this file when generating bitmap assets with `imagegen`.

## Prompt pattern

1. subject
2. expression and pose
3. material and lighting
4. palette
5. scene mood
6. composition constraints

## momo prompt ingredients

- big-eyed healing creature
- soft translucent jelly body
- creamy white with misty blue and lavender glow
- gentle floating pose
- lovable, quiet, reassuring

## Scene prompt guardrails

- keep negative space for UI overlays
- no text baked into the image
- no harsh shadows
- no aggressive action framing
- no generic sci-fi armor or fantasy weapons
