# Scene Direction

Use this file for Home art, mode-scene art, and growth-stage imagery.

## Home scene

- soft anchor composition
- quiet floating atmosphere
- enough breathing room for UI cards

## Mode scenes

- `light_drift`: light particles, upward motion, warm haze
- `cloud_breathing`: mist, roomy spacing, slow rings
- `bubble_release`: pressure bubbles, safe motion, calm close

## Growth stages

- `seed`: smaller, softer, simpler glow
- `bloom`: fuller body, brighter edge light
- `glow`: warmer halo, richer translucency, more presence

## Never do

- weapons
- armor
- angry face as the default
- neon gamer palette
- crowded backgrounds that fight the UI
