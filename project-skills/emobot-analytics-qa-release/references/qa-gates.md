# QA Gates

Use this file for test planning and ship blocking checks.

## Must-pass checks

- Mood Weather works end to end
- Treehole streams and ends cleanly
- each of the three mode scenes completes and records data
- Blind Box Lite draws and saves correctly
- growth page reflects new points and stage
- delete flow works
- report flow works
- privacy and age notice are reachable
- safety block interrupts risky chat correctly

## Regression focus

- route navigation
- offline or timeout fallback
- duplicate submissions
- repeated draw protection if the product limits draws
- growth point double-award bugs
