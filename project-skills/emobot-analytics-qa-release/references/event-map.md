# Event Map

Use this file when adding instrumentation.

## Core funnel

1. app open
2. Home render
3. check-in start
4. check-in complete
5. Treehole start
6. Treehole completion
7. mode scene start
8. mode scene completion
9. Blind Box draw
10. growth page open
11. helpfulness feedback

## Required dimensions

- device or anonymous user id
- session id when relevant
- mood type
- mode id
- card type
- completion status

## Priority metrics

- activation
- D7 retention
- helpfulness rate
- mode usage by mood
- AI cost per retained user
