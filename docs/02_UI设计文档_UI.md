# EmoBot — UI 设计规范文档
> 角色：UI 视觉设计师视角  
> 版本：v1.0 · 2026-04-17

---

## 1. 设计哲学 · Design Principles

1. **Soft Healing（柔和治愈）**：低饱和、大留白、柔光、无锐角。
2. **Living Aquarium（呼吸感）**：全局存在"活着"的粒子 / 光晕 / 水母浮动。
3. **Glass & Cream（玻璃与奶油）**：毛玻璃 Glassmorphism + 奶油质感背景层。
4. **One Hero（唯一主角）**：momo 是视觉 IP，所有页面都要为它让路。
5. **Anti-Stress**：拒绝红色警告、拒绝时间压迫视觉（倒计时 / 进度压力条）。

## 2. 色彩系统 · Color Token

### 2.1 基础色板（低饱和疗愈系）
| Token | HEX | 用途 |
|-------|-----|------|
| `--cream-50` | `#FDF8EF` | 主背景基底（奶油白）|
| `--cream-100` | `#F6EEDE` | 卡片底 |
| `--mist-blue-300` | `#AEC5E6` | 主视觉雾霾蓝 |
| `--mist-blue-500` | `#7896C9` | 品牌强调色 |
| `--lavender-300` | `#C8B8E0` | 薰衣草紫（AI 气泡）|
| `--lavender-500` | `#9E84CC` | 薰衣草紫（主按钮）|
| `--halo-white` | `rgba(255,255,255,.45)` | 玻璃拟态白 |
| `--glow-yellow` | `#FFE6B0` | 快乐模式光晕 |
| `--glow-red` | `#F4B6B6` | 愤怒模式光晕（柔和浅红）|

### 2.2 语义色
- 成功 / 确认：`--mist-blue-500`（不用绿，避免医疗感）
- 高危 / 熔断：`--lavender-500` 深调（不用红色，避免刺激）
- 文字主 `#1F2A44` / 次 `#5A6480` / 占位 `#A0A6B8`

### 2.3 情绪映射色（momo 动态变化）
```
低落: #7CA2D2 → 薰衣草紫
平静: #AEC5E6（默认）
愉悦: #FFD89B 渐变粉
愤怒: #E89A9A（柔化，不可饱和）
```

## 4. 间距 · 圆角 · 阴影

- **8 点栅格**：间距全用 4 的倍数（4/8/12/16/20/24/32）
- **圆角**：按钮 12、卡片 20、模态 28、胶囊 999
- **阴影（双层柔光）**：
  - 轻浮层：`0 4 12 rgba(120,150,201,.12)`
  - 浮起层：`0 12 32 rgba(158,132,204,.18)`
  - **禁用黑色硬阴影**

## 5. 玻璃拟态规范 · Glassmorphism Spec

```css
.glass-card {
  background: rgba(255,255,255,.45);
  backdrop-filter: blur(24px) saturate(160%);
  border: 1px solid rgba(255,255,255,.55);
  border-radius: 20px;
  box-shadow: 0 4 12 rgba(120,150,201,.12);
}
```
> iOS 支持 BackdropFilter 原生；Android < 12 需降级为半透明白 + 静态噪点纹理。

## 6. 粒子与光效系统

| 场景 | 粒子密度 | 动画 | FPS 预算 |
|------|---------|------|----------|
| 首页 | 15–20 | 漂浮 + 呼吸 | 60 |
| 树洞 | 8–12 | 发送消息时聚集 | 60 |
| 气象台 | 随情绪强度 5–30 | 拖滑杆触发涟漪 | 45+ |
| 闯关快乐 | 40–60 花瓣 | 通关时爆发 | 60 |
| 闯关低落 | 10 萤火 | 缓慢漂浮 | 30（节电）|
| 闯关愤怒 | 气泡粒子 | 破碎时迸发 | 60 |

**低端机降级策略**：CPU 占用 > 25% 时自动降为静态 PNG 背景。

## 7. 图标系统 · Iconography

- 基准尺寸 **24 × 24**（导航）/ **32 × 32**（功能入口）
- 风格：2pt 线条 + 圆角端点 + 可选径向渐变内阴影
- 绘制网格：6 格留白 + 20 内框
- 命名：`ic_[模块]_[状态]_[尺寸].svg`，如 `ic_treehole_active_24.svg`

## 8. momo 角色视觉规范（核心 IP）

### 8.1 三阶形态
| 阶段 | 体型 | 色彩 | 附件 |
|------|------|------|------|
| 细胞幼体 | 圆球 36px | 半透明白 | 无触手 |
| 水母进化体 | 伞盖 72px + 3 触手 | 雾霾蓝 / 动态情绪色 | 光晕 |
| 终极水晶水母 | 棱镜伞盖 96px + 6 触手 + 环 | 彩色棱镜 | 跟随手势 |

### 8.2 动作骨骼
- 使用 Lottie / Rive 骨骼 6 关节（伞盖 1、触手 3、光晕环 1、呼吸 1）
- 动作库 12 套：漂浮 / 点击弹跳 / 欢快 / 温柔 / 安抚 / 叹气 / 快速穿梭 / 击碎助攻 / 沉睡 / 唤醒 / 进化 / 首次见面

### 8.3 可变换参数
```
颜色 Hue：0–360
透明度：0.4–1.0
光晕半径：8–40
呼吸频率：0.5–1.5Hz
```

## 10. 动效规范 · Motion

| 类型 | 时长 | 缓动 |
|------|------|------|
| 按钮点击 | 120ms | `cubic-bezier(.4,.0,.2,1)` |
| 卡片入场 | 320ms | `cubic-bezier(.2,.8,.2,1)` |
| 页面切换 | 400ms 共享元素过渡 | ease-in-out |
| momo 呼吸 | 2000ms 循环 | sine |
| 粒子漂浮 | 6000ms 循环 | linear |
| 通关庆祝 | 1200ms | elastic |

## 11. 暗色模式策略

- **默认不提供暗黑模式**，而是「夜间护眼模式」：整体调到雾霾蓝 + 薰衣草紫深调，保留奶油感。
- 原因：治愈系产品若全黑化会削弱"光的希望"隐喻。
- 触发：系统暗黑 + 晚上 22:00 后提示一次。

## 12. 无障碍 · Accessibility

- 对比度最低 4.5:1（气泡正文已验证）
- 支持 iOS Dynamic Type 字号放大到 200%
- 所有动画可在"减少动态效果"系统设置下停用
- 色盲安全：情绪颜色必须同时附带**图标（晴/云/雨）+ 情绪描述文字**

## 13. 切图与资产交付规范

- 矢量优先：图标一律 SVG，复杂插画用 Lottie / Rive
- 位图 @2x @3x，PNG-8 压缩
- momo 三态使用 Rive 单文件 + 状态机
- 设计稿工具：Figma，组件命名 `EB/Component/State`
- Design Token 以 JSON 导出供前端 `design-tokens` 库消费
